Isaiah Visualizations: Custom Properties

On this page I define some custom properties I'm using in my Simile Exhibits on this site. I've created a timeline of relevant historical events in the book of Isaiah as well as a map. These visualizations were created to supplement course materials for Ex204 Exposition of Isaiah at Foundation Baptist College, Edmonton, AB.

Properties used in my map.

"scripture" (array, string)

Contains either a string or an array of strings containing biblical references for an item. If well-formed these references are linked to Bible browsing sites via bib.ly. Examples below.

String

	"scripture" : [ "Isa 21:14" ],

Array

	"scripture" : [ "Isa 11:11", "Isa 18:1", "Isa 20:3", "Isa 20:5", "Isa 37:9", "Isa 43:3", "Isa 45:14" ],
